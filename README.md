# Simple Discord Music Queue Commands
Kumpulan command `m!p` untuk membantu membuat antrian musik di bot musik Discord.

## Fitur
- Format command sederhana: `m!p <judul>`
- Contoh playlist berdasarkan artis/album
- Mudah di-copy untuk kebutuhan server musik

## Contoh Penggunaan
m!p Slow Dancing in the Dark
m!p Sanctuary
m!p Will He
